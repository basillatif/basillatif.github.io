Structure of A Request

Endpoint - Method - Headers - Body

curl + [Type of Request i.e. '-X' or '--request'] + [METHOD] + [URL] + [DATA]

Example Endpoint of GitHub API

Endpoint: https://api.github.com

5 Possible Methods

Get/Post/Put/Patch/Delete

Post Method:

Example: curl -X POST https://api.github.com/user/repos

Header Example

the "-H" option lets you specify a header

curl -H "Content-Type: application/json" https://api.github.com

Body Example

Use "-d." For example: curl -X POST -d property1=value1
